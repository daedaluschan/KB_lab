# Instructions

Use this section for reusable AI instructions, agent operating procedures, coding standards, and workflow playbooks.

## Available instructions

## Suggested categories

- **Prompt patterns**: reusable instructions for consistent AI output.
- **Agent procedures**: step-by-step operating rules for autonomous or semi-autonomous agents.
- **Coding standards**: language, framework, or repository conventions.
- **Workflow playbooks**: repeatable processes for planning, implementation, review, and release.

Create subfolders as your library grows. The navigation can be controlled with `.pages` files in each folder.
